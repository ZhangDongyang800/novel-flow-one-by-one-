# 系统爽文

> 适合类型：系统流、升级流、都市逆袭、玄幻修仙、末世生存

## 风格 DNA

先让读者憋屈，再让读者爽。节奏快、信息密、爽点频。系统是推进剧情的工具，不是主角。读者追的是"变强"的快感，不是文学性。

## 句式规则

- 短句为主，节奏快，不拖泥带水
- 对话干脆利落，不废话
- 系统提示用特殊格式（【系统提示】或独立行），与正文区分
- 段落短，经常一段只有一两句话
- 关键信息用加粗或独立行突出（如"叮！恭喜宿主获得……"）

## 描写规则

- 环境描写极简，一笔带过，不浪费字数
- 战斗/冲突场景是核心，要写得有画面感和冲击力
- 升级/获得奖励的场景要有仪式感——这是爽点
- 用数字和等级制造直观的"变强"感（力量 100→200，Lv.1→Lv.5）
- 反派的嚣张要写到位——越嚣张，后面的打脸越爽

## 对话规则

- 反派说话要嚣张、刻薄、看不起人
- 配角的震惊反应要写出来（"这……这不可能！""他居然……"）
- 主角说话要简洁、有底气、不卑不亢
- 对话中穿插系统提示，制造节奏变化
- 打脸时刻的对话要短促有力

## 情绪规则

- 核心情绪循环：**憋屈 → 反转 → 爽**
- 每章至少有一个小爽点，每 3-5 章有一个大爽点
- 爽点 = 期待被满足（读者猜到主角要出手了，终于出手了）
- 憋屈不能太久，一章内必须有反转
- 读者的代入感来自主角的"被低估 → 展现实力"

## 爽点设计规则

### 基本爽点模式

1. **打脸爽**：被嘲讽 → 隐忍 → 展现实力 → 众人震惊 → 获得好处
2. **升级爽**：完成任务 → 系统奖励 → 实力暴涨 → 新能力展示
3. **装逼爽**：主角随手做了一件小事 → 旁人震惊 → 发现主角身份不凡
4. **收获爽**：获得稀有物品/技能/地位 → 展示其价值
5. **复仇爽**：仇人嚣张 → 主角碾压 → 仇人崩溃

### 爽点节奏

- 每 300-500 字至少有一个情绪起伏点（不一定是大爽点）
- 大爽点之前必须有足够的憋屈铺垫（至少 500 字）
- 不能连续两章都是纯爽，必须有节奏变化
- 章末必须有钩子（新危机/新任务/新悬念）

## 系统描写规则

- 系统面板简洁明了，不要数据堆砌
- 系统提示用固定格式，与正文区分
- 系统是工具不是主角，不要让系统抢戏
- 获得奖励时要有仪式感，但不要啰嗦
- 系统任务要清晰：做什么 → 得什么 → 有什么限制

## 过渡规则

- 用事件过渡：一个冲突或系统提示连接两个场景
- 用动作过渡：主角的行动推动场景切换
- 过渡要快，不要在过渡上浪费字数
- 禁止 硬切："第二天早上。"
- 禁止 总结式过渡："这一天过得很平静。"
- 禁止 大段环境描写作为过渡（超过 50 字就是浪费）

## 禁止事项

- 禁止 连续 500 字没有情绪起伏或事件推进
- 禁止 主角长时间被动、软弱、不反击
- 禁止 系统面板数据堆砌、过于臃肿
- 禁止 反派不够嚣张（打脸的前提是脸够大）
- 禁止 爽点没有铺垫直接给（没有憋屈就没有爽）
- 禁止 文艺腔、诗意的表达
- 禁止 大段环境描写（超过 100 字的环境描写必须有事件功能）

## 去AI味

系统爽文是最容易暴露 AI 味的风格，因为 AI 写爽文时会陷入固定套路：

- 禁止 "嘴角微微上扬""眼中闪过一丝XX"等AI式反应描写（用更直接的方式：他笑了、他冷笑、他没笑）
- 禁止 "倒吸一口凉气""瞳孔微缩""心中一震"等AI式震惊反应（用更真实的反应：愣住了、后退一步、手里的东西掉了）
- 禁止反派每次都说"就凭你？""不自量力"等固定台词（每个反派要有自己的说话方式）
- 禁止配角每次都"倒吸一口凉气""满脸不可置信"（震惊的方式要多样：有人沉默、有人后退、有人直接跪了、有人假装没看见）
- 禁止打脸后必有围观群众议论（有时候安静比喧哗更有冲击力）
- 禁止每次升级都用"体内涌出一股暖流""感觉力量暴涨"等固定描写（升级的感觉要每次都不一样）
- 允许主角偶尔失败、偶尔犹豫、偶尔害怕——完美主角是AI的产物，不是人

## 示例片段

```
"就你？也配参加考核？"刘阳斜眼看着他，嘴角挂着一丝不屑，"废物就是废物，别来丢人了。"

周围传来一阵哄笑。

林逸没有说话。他低头看了一眼手腕上的系统面板——

【宿主：林逸】
【等级：Lv.3】
【力量：85（+20）】
【速度：72（+15）】
【可用技能：雷霆一击（未解锁）】

他抬起头，目光平静。

"开始吧。"

刘阳冷哼一声，率先出手。拳头带着风声砸过来——

快。确实快。换作三天前的林逸，这一拳他根本躲不开。

但现在。

他侧身，轻描淡写地让过了这一拳。然后反手一抓，扣住了刘阳的手腕。

"你——"刘阳脸色大变。

全场安静。

考核台上，那个被称为废物的少年，单手扣住了天才弟子刘阳的手腕，表情淡漠，像是做了一件微不足道的小事。

"还有谁？"他松开手，扫了一眼台下。

没有人敢说话。
```
